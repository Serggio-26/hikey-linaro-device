# hikey-linaro-device
Cloned from https://android.googlesource.com/device/linaro/hikey.git

BSP for Android 9.0 for [**hikey960**](https://www.96boards.org/product/hikey960/) board.
Includes hext customisation:
- Use Linux kernel source instead of prebuilt kernel
- Add command to build vendor specific SW in *vendor* directory
